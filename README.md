#Components

--> SubmittButton :

-   Muestra el texto "ok".
-   ejecuta la función que recibe.(recoger y enviar el valor de input)

--> Input :

-   recibe/recoge el valor que el usuario escribe: una sola letra.
    --limite de 1 letra.

--> App:

-   función para comprobar que el valor recibido no se repita.
-   renderiza el input, el boton, el muñeco, lista de palabras usadas, los divs de palabras.
-   tiene el array de inicial (de palabras o la palabra con sus letras--tbc)

--> Hangman:

-   recibe la acción de pintarse (o no).

--> letterContainer:

-   muestra un valor recibido.
-   recibe la indicación de mostrar un valor.

--> wrongAnswerContainer:

-   muestra valores recibidos (no se corresponden con la verificación)
